# フルコポ — Fool or Corporation

## フォルダ構成

```
fulcopo/
├── index.html          ← メインHTMLファイル（これをブラウザで開く）
├── css/
│   └── style.css       ← 全スタイル
├── js/
│   ├── data.js         ← カード・役職データ、シャッフル関数
│   └── game.js         ← ゲームロジック本体
└── assets/             ← 画像・音声ファイル置き場（今後使う場合）
```

## 遊び方（起動方法）

### ローカルで遊ぶ
1. フォルダをそのまま手元に置く
2. `index.html` をブラウザで開く
3. 遊べます！（サーバー不要）

### 公開してみんなで遊ぶ（おすすめ）

#### GitHub Pages（無料・簡単）
1. GitHubに新しいリポジトリを作成
2. `fulcopo/` の中身をすべてアップロード
3. Settings → Pages → Branch: main → Save
4. URLが発行されたらそこでプレイ可能

#### Netlify（無料・ドラッグ＆ドロップ）
1. https://netlify.com にアクセス
2. `fulcopo/` フォルダをドラッグ＆ドロップ
3. 自動でURLが発行される

## ゲームルール

- 6人用カードゲーム（プレイヤー1人 + CPU5人）
- 全員に役職カードが1枚配られる（秘密）
- トランプ1デッキ（54枚）を9枚ずつ配布
- 毎ラウンド1枚ずつ出し合い、最強カードを出した1人が勝者
- 同じ強さのカードが被ったらそのラウンドはキャンセル
- ジョーカーはもう1枚見せて、そのカードとして機能する
- 誰かの手札がなくなったらゲームセット
- 最多ラウンド勝者が優勝
